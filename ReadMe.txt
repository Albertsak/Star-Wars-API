Proyecto API Star Wars

Descripción:

Breve Api del universo cinematográfico de Star Wars, donde puedes obtener la información y una imagen de cada personaje importante dentro de la saga.

URL: https://swapi.py4e.com/api/people/  

URL de prueba:
https://swapi.py4e.com/api/people/?page=${pagina}

{pagina} es el número de cada personaje. Ejemplo: https://swapi.py4e.com/api/people = Página del personaje Luke Skylewalker.

Problemas encontrados:
-Un personaje no muestra correctamente los datos ni la imagen.
